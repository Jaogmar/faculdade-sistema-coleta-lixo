# Sistema de Coleta de Lixo

Projeto academico desenvolvido para a faculdade, com foco em simular um sistema de apoio a coleta de lixo urbano.

## Sobre o projeto

Este sistema foi criado como parte de um trabalho de faculdade, seguindo os requisitos definidos pela disciplina.

Atualmente, o projeto funciona apenas no frontend (interface), sem backend implementado. Isso significa que os dados sao tratados no proprio cliente para fins de demonstracao e validacao da proposta.

## Objetivos

- Apresentar uma interface para consulta e acompanhamento de pontos/coletas.
- Simular fluxos de uso relacionados a protocolos, notificacoes e operacao de coleta.
- Atender aos requisitos funcionais solicitados no trabalho academico.
- Servir como base para evolucao futura com integracao de backend.

## Escopo academico

Este repositorio representa uma entrega de faculdade. A modelagem e as funcionalidades foram construidas para cumprir o escopo pedido no projeto da disciplina.
